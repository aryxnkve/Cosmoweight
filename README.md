# Cosmic Weight Calculator 🪐

Welcome to the **Cosmic Weight Calculator**! This is an interactive web application built with React and TypeScript that calculates your weight across different celestial bodies—planets, moons, and stars—using real physics and gravity data.

## Features ✨
- **Accurate Gravity Physics**: Calculates relative weight based on surface gravity data.

- **Educational Profiles**: Learn facts and details about every celestial body included.

## How to Download and Run This Project 🚀

You can easily run this project on your own device using either **Node.js** or **Docker**.

### Method 1: The Standard Way (Requires Node.js)
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

1. **Download the code**: Either click **Code -> Download ZIP** on GitHub and extract it, or use Git:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
   ```
2. **Open the folder**: Open your terminal (or Command Prompt/PowerShell) and navigate into the project directory (make sure the folder name matches):
   ```bash
   cd Planetary_weight
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Start the Development Server**:
   ```bash
   npm run dev
   ```
5. **Open in Browser**: The terminal will display a local link (usually `http://localhost:5173`). Click it or copy-paste it into your web browser to view the app!

### Method 2: The Docker Way (Requires Docker)
If you prefer not to install Node.js and already have [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed:

1. Download the code and open your terminal in the project folder (as shown in steps 1 and 2 above).
2. Run this command:
   ```bash
   docker compose up --build
   ```
3. Once running, open your browser and go to `http://localhost:8080`.
